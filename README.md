# Sensor de Aproximação

## Objetivo do Projeto
Este projeto foi desenvolvido durante as aulas de IIoT do curso de Desenvolvimento de Sistemas no SENAI, com o objetivo de aprendermos sobre o sensor de aproximação. O sensor consegue detectar se algo se aproximou, enviando uma mensagem informando se o ambiente está seguro ou se há algum intruso detectado.

## Funcionalidades
**Verificação de aproximação** - O sensor faz verificações a cada 3 segundos. <br>
**Detecção de presença** - O sensor retornará um valor de distância: quando o objeto estiver a 10 cm ou menos, será considerado que há um intruso próximo. <br>
**Alerta ao usuário** <br>
- Se um objeto ou pessoa estiver a 10 cm ou menos do sensor, um LED ficará aceso e será exibida a mensagem: "INTRUSO DETECTADO!" <br>
- Caso contrário, o LED ficará apagado e será exibida a mensagem: "Ambiente seguro."
